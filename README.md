# TODO APPLICATION

This app helps to save your todos in the server

#### Instructions

1. Download the sourse code
```bash
git clone https://github.com/00010174/todos.git
```

or just download the *.zip file

2. Install dependincies
```bash
npm install
```

3. Run the applicaiton
```bash
node app
```

#### Dependencies list
- express
- pug
